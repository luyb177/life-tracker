# life-tracker
个人生活账本

## 目标
### 首要目标
- 记录每天做了什么（可以多次记录，随时随地记录）
- 记录每天花了多少钱（分为早饭，午饭，晚饭，杂项，以及一些经常性重复的可以自己预设的事情，比如吹头发等内容；这每一笔花销都可以记录）
### 次要目标
- AI 会定时总结今日的生活，给出一些建议（比如今天吃了什么，花了多少钱，睡了多久，运动了多久等内容）
- AI 会定时总结本周的生活，给出一些建议（比如本周吃了什么，花了多少钱，睡了多久，运动了多久等内容）
- AI 会定时总结本月的生活，给出一些建议（比如本月吃了什么，花了多少钱，睡了多久，运动了多久等内容）
- AI 会定时总结今年的生活，给出一些建议（比如今年吃了什么，花了多少钱，睡了多久，运动了多久等内容）
- AI 会定时总结人生的生活，给出一些建议（比如人生吃了什么，花了多少钱，睡了多久，运动了多久等内容）
